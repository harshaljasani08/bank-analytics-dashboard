# Bank Analytics Dashboard – Credit & Debit Analysis

## Project Overview
This project analyzes banking transaction data to understand customer credit and debit behavior. The goal is to generate actionable insights using dashboards and structured queries.

## Business Objectives
- Analyze credit vs debit transaction trends
- Identify high-value customers and transaction patterns
- Compare transaction behavior across branches and time periods
- Build dashboards for decision-making support

## Key KPIs
- Total Credit Amount
- Total Debit Amount
- Credit vs Debit Ratio
- Transaction Volume
- Average Transaction Value
- Customer-wise and Branch-wise Performance

## Tools & Technologies
- **Excel** – Dashboard creation, Pivot Tables, KPI analysis  
- **SQL** – Data querying, filtering, aggregation  
- **Power BI** – Interactive data visualization  
- **Tableau** – Visual analytics and trend analysis  

## Repository Contents
- `Bank_Debit_and_Credit_Dashboard.xlsx` – Excel dashboard  
- `SQL_credit_Debit_Queries.sql` – SQL queries used for analysis  
- `powerbi_bank_analytics_dashboard.pbix` – Power BI dashboard  
- `TABLEAU_credit_debit_dashboard.twbx` – Tableau dashboard
